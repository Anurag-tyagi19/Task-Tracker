# Task Tracker

A minimal Task Tracker built with the MERN stack (MongoDB, Express, React, Node.js).

## Prerequisites
- Node.js installed
- A MongoDB connection string (set up in `backend/.env`)

## 1. Backend Setup (API Server)
The backend runs on `http://localhost:5000`.

1. Open a terminal and navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm run dev
   ```

## 2. Frontend Setup (React App)
The frontend runs on `http://localhost:5173`.

1. Open a **new** terminal and navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm run dev
   ```

## Usage
Once both servers are running, open your browser and go to [http://localhost:5173](http://localhost:5173) to manage your tasks!
